# User Stories

#### User
* DONE Sign up, in, out
* DONE Email Auth

* Facebook Auth
* I can C(RUD) my account

#### Codebin
* DONE I can CRUD a Codebin live.

* I can text search for Codebins
* I can text search for my codebins

#### Pages
* Home
* Search
* Dashboard
* Settings
* Editor
* DONE Sign up
* DONE Sign in


#### Bonus
* I can share a codebin
* I can like a bin
* I can comment on a bin
